## ğŸ“‹ How to Use

### First Time Setup
1. Download and install the APK
2. Tap "Create Account"
3. Enter your full name, email, and password
4. Agree to terms and register
5. Welcome screen appears
6. Start adding tasks!

### Adding Tasks
1. Tap the "+ Add Task" button or use the drawer menu
2. Fill in title, description, subject
3. Set deadline and priority
4. Save and view on dashboard

### Managing Tasks
- View all tasks in "All Tasks" tab
- Filter by subject or status
- Swipe to refresh
- Tap to edit
- Mark as complete

### Account Settings
- Edit profile name and photo
- Change password
- View task statistics
- App settings and notifications
