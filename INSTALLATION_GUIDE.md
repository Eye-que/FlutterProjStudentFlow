### Step 2: Enable Unknown Sources
On your Android device:
1. Go to **Settings**
2. Navigate to **Security** (or **Privacy** on some devices)
3. Enable **"Install from Unknown Sources"** or **"Allow installation of unknown apps"**
4. Select your browser or file manager

### Step 3: Install the APK
1. Open the downloaded `app-release.apk` file
2. Tap **"Install"**
3. Wait for installation to complete
4. Tap **"Open"** to launch the app

## ğŸ”§ Troubleshooting

### Issue: "App not installed" Error

**Solution:**
1. Make sure "Unknown Sources" is enabled
2. Check available storage space
3. Try downloading the APK again
4. Restart your device and try again

### Issue: Can't Create Account

**Possible Causes:**
- No internet connection
- Email already registered
- Weak password (min. 6 characters)
- Firebase configuration issue

**Solution:**
1. Check internet connection
2. Try logging in instead if email exists
3. Use a stronger password
4. Try a different email

### Issue: App Crashes on Launch

**Solution:**
1. Clear app cache:
